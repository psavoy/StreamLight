# Changelog

## StreamLight 0.2.0

### 1.0.0 (3/13/2026)

- Update to version 1.0.0. I think it is time to finally have the
  versioning reflect that this package is no longer in development
  status. To go along with that I have updated and revised the tutorials
  and gone through things to make sure everything is functioning.

### 0.2.0 (5/18/2021)

- Update to version 0.2.0 includes a massive update to include the new
  **aqua_light** function which generates estimates of
  photosynthetically active radiation at the benthic surface! The
  details and application of **aqua_light** are detailed in [Savoy &
  Harvey.
  (2021a)](https://agupubs.onlinelibrary.wiley.com/doi/abs/10.1029/2020GL092149),
  and the modeled estimates from this paper are available in a
  [ScienceBase data release (Savoy & Harvey,
  2021b)](https://www.sciencebase.gov/catalog/item/5f974adfd34e198cb77db168).
