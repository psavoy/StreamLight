# Package index

## All functions

- [`RT_CN_1998()`](https://psavoy.github.io/StreamLight/reference/RT_CN_1998.md)
  : Calculate below canopy PAR
- [`SHADE2()`](https://psavoy.github.io/StreamLight/reference/SHADE2.md)
  : SHADE2 model from Li et al. (2012) Modeled riparian stream shading:
  Agreement with field measurements and sensitivity to riparian
  conditions
- [`SHADE2_AL()`](https://psavoy.github.io/StreamLight/reference/SHADE2_AL.md)
  : Modified version of the SHADE2 model
- [`absorption()`](https://psavoy.github.io/StreamLight/reference/absorption.md)
  : Calculate light absorption in pure water
- [`aqua_light()`](https://psavoy.github.io/StreamLight/reference/aqua_light.md)
  : Model to predict photosynthetically active radiation at the benthic
  surface.
- [`azimuth_adj()`](https://psavoy.github.io/StreamLight/reference/azimuth_adj.md)
  : Helper function for determining the correct solar azimuth
- [`deg2rad()`](https://psavoy.github.io/StreamLight/reference/deg2rad.md)
  : Function for converting from degrees to radians
- [`predict_transmission()`](https://psavoy.github.io/StreamLight/reference/predict_transmission.md)
  : Calculate light transmission as a function of depth and and the
  irradiance attenuation coefficient
- [`rad2deg()`](https://psavoy.github.io/StreamLight/reference/rad2deg.md)
  : Function for converting from radians to degrees
- [`shade_calc()`](https://psavoy.github.io/StreamLight/reference/shade_calc.md)
  : Calculating the percent of the wetted width shaded by banks and
  vegetation
- [`shade_calc_AL()`](https://psavoy.github.io/StreamLight/reference/shade_calc_AL.md)
  : Calculating the percent of the wetted width shaded by banks and
  vegetation modified for use with aqua_light
- [`solar_c()`](https://psavoy.github.io/StreamLight/reference/solar_c.md)
  : Calculates solar geometry for use in the SHADE2 model
- [`solar_geo_calc()`](https://psavoy.github.io/StreamLight/reference/solar_geo_calc.md)
  : Calculates solar geometry
- [`stream_light()`](https://psavoy.github.io/StreamLight/reference/stream_light.md)
  : Model to predict light at the stream surface
- [`surface_reflection()`](https://psavoy.github.io/StreamLight/reference/surface_reflection.md)
  : Calculate reflection from the water surface
