# Calculate reflection from the water surface

This function calculates reflection on the water surface based on solar
geometry following Kirk (2011) Light and photosynthesis in aquatic
ecosystems.

## Usage

``` r
surface_reflection(driver_file, solar_geo)
```

## Arguments

- solar_geo:

  Solar geometry, calculated from solar_geo_calc.R

- driver:

  The site driver file
