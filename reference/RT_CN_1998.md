# Calculate below canopy PAR

This function calculates below canopy PAR

References \#'

- Campbell & Norman (1998) An introduction to Environmental biophysics
  (abbr C&N (1998))

- Spitters et al. (1986) Separating the diffuse and direct component of
  global radiation and its implications for modeling canopy
  photosynthesis: Part I components of incoming radiation

- Goudriaan (1977)

## Usage

``` r
RT_CN_1998(driver_file, solar_geo, x_LAD, ...)
```

## Arguments

- driver_file:

  The model driver file

- solar_geo:

  Solar geometry, calculated from solar_geo_calc.R

- x_LAD:

  Leaf angle distribution, default = 1

## Value

Returns a time series of below canopy PAR
